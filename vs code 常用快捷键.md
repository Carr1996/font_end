## vs code 常用快捷键

- 命令面板：F1 /  Ctrl+Shift+P

- 代码行缩进：Ctrl + [ 、Ctrl + ]

- 文件切换：Ctrl + Tab

- 重命名：F2

- 全屏：F11

- 上下移动一行：Alt + Up / Alt + Down

- 代码格式化：Shift＋Alt + F

- 直接打开浏览器：Alt+B

- 在上方或下方复制一行：Shift+Alt+Up/Down 

- 在上方或下方添加光标：Ctrl+Alt+Up/Down

- 转到行首/行尾：Home / End

- 转到文件头/文件尾：Ctrl + Home / Ctrl + End

- 资源管理器： Ctrl+Shift+E

- 搜索： Ctrl+Shift+F

- 拓展：Ctrl+Shift+X

- 侧边栏：Ctrl+B

- 新建文件：Ctrl+N

- 新开一个窗口：Ctrl+Shitf+N

- 切出新的编辑器：Ctrl+\

- 编辑器间切换：Ctrl+1/2/3

- 在当前行下边插入一行 :Ctrl+Enter

- 在当前行上方插入一行 ：Ctrl+Shift+Enter

- 选择从光标到行尾： Shift+End

- 选择从行首到光标处： Shift+Home

- 查找 ：Ctrl+F

- 查找替换：Ctrl+H

- 在一组标签外面包一个  选中Alt+W
